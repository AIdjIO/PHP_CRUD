# MySQL commands needed
## INSERT (CREATE)
```SQL
INSERT INTO table ( col1, col2 ) VALUES ('val1', 'val2' )
```

## SELECT (READ)
```SQL
SELECT * FROM table WHERE col1 = 'val1' AND col2 = 'val2'
```

## UPDATE
```SQL
UPDATE table set col1 = 'val1', col2 = 'val2' WHERE col3 = 'val3
```

## DELETE
```SQL
DELETE FROM table WHERE col1 = 'val1'
```
